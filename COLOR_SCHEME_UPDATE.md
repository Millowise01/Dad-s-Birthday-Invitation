## 💡 Customization

Want to change colors again? Edit `frontend/src/App.css`:

```css
:root {
  --gold: #87CEEB;        /* Change this to your color */
  --dark-gold: #4A90E2;   /* Darker version for hover */
  --black: #000000;       /* Text color */
  --white: #ffffff;       /* Background */
  --gray: #f8f9fa;        /* Light background */
  --dark-gray: #333333;   /* Secondary text */
}
```

### Popular Alternative Color Schemes:

## 🔄 Reverting to Original Colors

If you want the original gold and black theme back:

```css
:root {
  --gold: #d4af37;        /* Gold */
  --dark-gold: #b8941f;   /* Dark gold */
  --black: #1a1a1a;       /* Dark background */
  --white: #ffffff;
  --gray: #f5f5f5;
  --dark-gray: #333;
}
```

And update the hero section background:
```css
.hero {
  background: linear-gradient(135deg, #1a1a1a 0%, #333 100%);
  color: #ffffff;
}
```
